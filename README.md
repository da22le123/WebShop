# WebShop
A simple copyshop where users can view a product catalogue and add products to their cart

## How to run

### .env

**Backend**
- PORT
- JWT_SECRET

**Frontend**

- VITE_API_URL

### Start'em

**Backend** 
- `cd backend`
- `npm install`
- `npm run start``

**Frontend**
- `cd frontend`
- `npm intall`
- `npm run dev`

## Tests

To run backend tests execute `npm run test`.
